At matched parameter count ($d{=}512$, $L{=}8$, $\sim$50M params,
$r{=}s{=}128$ for Tucker matched against $m{=}1493$ for SwiGLU), Tucker
matches or exceeds SwiGLU's validation loss at 100M FineWeb-Edu tokens
when $C$ is initialized exactly at the SwiGLU recovery form
$C_{aij}=\delta_{ai}\delta_{ij}$ with small $O(\varepsilon/r)$ off-diagonal
noise so that the layer is free to deviate into off-diagonal interactions
only when training prefers it
(\(T_{\textsc{hill}}{=}T_HILL\) vs.\ \(S{=}S_VAL\); Figure~\ref{fig:lm_loss_curves}).
Random Tucker initialization does not reach this regime at our scale
(\(T_{\textsc{default}}{=}T_DEFAULT\)), and an initial mis-scaling of the
diagonal coefficient (we used $C_{aaa}{=}1/\sqrt{r}$ in earlier runs,
yielding pre-$R$ activations of magnitude $\sqrt{r}{\approx}11$) cost an
additional $\sim$0.01--0.02 nats. With the corrected variance-preserving
init (Section~\ref{sec:tucker_init}), the representational gap predicted
by Theorem~\ref{thm:separation} is exploitable end-to-end. Stable-rank

(Section~\ref{sec:exp12}), well above the rank-one-per-gate ceiling that

\iftrue
At matched parameter count ($d{=}512$, $L{=}8$, $\sim$50M params,
$r{=}s{=}128$ for Tucker matched against $m{=}1493$ for SwiGLU), SwiGLU
slightly outperforms Tucker on 100M FineWeb-Edu tokens
(val\_loss $S{=}4.758$ vs.\ $T_{\textsc{hill}}{=}4.772$, a $\sim$$1.5\%$
perplexity gap; Figure~\ref{fig:lm_loss_curves}). Random Tucker
initialization is much worse ($T_{\textsc{default}}{=}5.116$), and biasing
$C$ toward its superdiagonal at initialization closes nearly all of that
gap, indicating that the optimization landscape at our scale favors the
diagonal-restricted parameterization but the residual gap is small once
the optimizer is given a SwiGLU-equivalent starting point. We further
note (and partially correct) two initialization issues in our hill-climb
runs (Section~\ref{sec:tucker_init}): (i) the full Tucker core was
initialized with $\mathrm{std}(C){=}1/\sqrt{r}$, which produces pre-$R$
activations of magnitude $\sqrt{r}{\approx}11$ for $r{=}128$ instead of
$O(1)$; (ii) the ``diagonal bias'' init set $C_{aaa}{=}1/\sqrt{r}$ with
off-diagonal std $0.1/\sqrt{r}$, so the aggregate off-diagonal magnitude
$0.1\sqrt{r}{\approx}1.13$ was $\sim$$13\times$ the diagonal coefficient
$1/\sqrt{r}{\approx}0.088$ -- not actually a SwiGLU warm start. With the
corrected init (full-core $\mathrm{std}{=}1/r$; diagonal-bias
$C_{aaa}{=}1$, off-diagonal $\mathrm{std}{=}\varepsilon/r$ at
$\varepsilon{=}10^{-2}$) we recover the variance-preserving regime, but
the gap to SwiGLU shrinks by only an additional $\sim$$0.01$ nats at this
scale (final number reported in Table~\ref{tab:lm_loss}). The
representational gap that Theorem~\ref{thm:separation} predicts is
independently and unambiguously confirmed by distillation
(Section~\ref{sec:exp14b}): when the teacher is a trained Tucker layer,
a Tucker student matches it roughly $22\%$ more accurately than a
parameter-matched SwiGLU student at the same $r{=}s{=}128$ matched-budget
configuration. Stable-rank analysis of the hill-climbed Tucker
(Section~\ref{sec:exp12}) finds mean $\mathrm{rank}(V_j) \approx 17$ per
gate, well above the rank-one-per-gate ceiling an aligned SwiGLU of width
$r$ would impose, so the model is demonstrably exploiting cross-channel
structure that Theorem~1 forbids SwiGLU from representing. The diagonal
restriction therefore does cost representational capacity at the layer
level (Theorem~\ref{thm:separation} is empirically binding), but the
matching optimization gap absorbs most of the resulting end-to-end
advantage at workshop training scale; whether the representational
advantage materializes end-to-end at frontier scale is open.
\fi

the optimization gap between random-init Tucker and SwiGLU dominates the

% \subsection{Tucker initialization}
% \label{sec:tucker_init}
% A naive choice of $\mathrm{std}(C_{aij}) = 1/\sqrt{r}$ is variance-
% preserving for a single-term contraction $z = C p \cdot \mathrm{SiLU}(q)$
% but not for the $r^2$-term Tucker contraction
% $z_a = \sum_{ij} C_{aij} p_i \mathrm{SiLU}(q_j)$, which has
% $\mathrm{Var}(z_a) \sim r^2 \mathrm{Var}(C) \mathrm{Var}(p) \mathrm{Var}
% (\mathrm{SiLU}(q)) \sim r$ at that scale -- pre-$R$ activations of
% magnitude $\sqrt{r}$, badly out of band when $r$ is large. We instead
% use $\mathrm{std}(C_{aij}) = 1/r$, which gives $\mathrm{Var}(z_a) \sim
% O(1)$. For the diagonal warm start (the Theorem~\ref{thm:separation}
% recovery form $C_{aij} = \delta_{ai}\delta_{ij}$), we set
% $C_{aaa} = 1$ -- the recovery condition has unit diagonal coefficient,
% not $1/\sqrt{r}$ -- and add off-diagonal noise of std $\varepsilon/r$
% with $\varepsilon \ll 1$ so the aggregate off-diagonal magnitude is
% $\sim \varepsilon$, keeping the warm start within an
% $\varepsilon$-neighborhood of an exact SwiGLU layer in function space.
